# AI Construction Video System Production Stack

## Summary

The production system uses a fixed tool stack.

Each tool has a dedicated responsibility.

Avoid unnecessary tools.

The objective is predictable and scalable video production.

# Tool 1 — ChatGPT

## Purpose

Acts as the Architect.

Responsible for:

- Concept creation
- Scene planning
- Image prompt creation
- Video prompt creation
- Workflow design
- Troubleshooting

## Inputs

- Research database
- Concept database
- Existing templates

## Outputs

- Scene breakdown
- Image prompts
- Video prompts

## Rules

Never generate images directly.

Always focus on planning and prompt engineering.

---

# Tool 2 — Image Generation

## Purpose

Create every scene as an image before generating video.

Images are the foundation.

Weak images produce weak videos.

## Responsibilities

- Environment creation
- Consistency maintenance
- Start Frame creation
- End Frame creation

## Rules

Use Image-to-Image workflow.

Never regenerate scenes from scratch.

Maintain:

- Camera position
- Lighting
- Environment
- Scale

---

# Tool 3 — Video Generation

## Purpose

Create motion between Start Frame and End Frame.

The model should not invent logic.

The workflow should define the transformation.

## Responsibilities

- Timelapse motion
- Construction actions
- Material movement
- Physical progression

## Rules

Every video scene must have:

- Start Frame
- End Frame
- Worker
- Timelapse
- ASMR

---

# Tool 4 — Editing

## Purpose

Combine generated clips into a finished short-form video.

## Responsibilities

- Clip assembly
- Trimming
- Speed adjustment
- Hook optimization
- Final reveal optimization

## Rules

Avoid:

- Excessive effects
- Heavy transitions
- Cinematic distractions

Focus on:

- Clarity
- Progression
- Retention

Before production:

- Concept selected
- Scene breakdown completed

- Hook visible in first seconds
- Transformation clear
- Reveal satisfying
