# AI Video Prompt Format: Hidden Nature Shelter Build

## 1. Core Concept

The video concept is:

> An impossible hidden location in nature is transformed into a livable shelter or tiny house.

The key emotion is surprise.

The viewer should first think:

> "There is no way someone could live here."

Then, through a clear construction workflow, the place gradually becomes a believable, cozy living space.

This is not just a DIY construction video.
It is a transformation video:

> Impossible natural location → hidden human shelter → final cinematic reveal

---

## 2. Content Principles

### 2.1 Location Requirement

The location must feel unexpected, hidden, and naturally impossible.

Good examples:

- A crack inside a giant cliff
- A hollow inside a massive ancient tree
- A cave behind a waterfall
- A gap between huge boulders
- An underground stone chamber in a forest
- A hidden cave under tree roots
- A cliffside hole overlooking a valley
- A moss-covered rock wall with a secret entrance

The location should make the viewer feel:

- "Why is there a house here?"
- "How did they build this?"
- "I want to see the final result."

---

## 3. Standard Workflow Structure

Use this workflow for every video.

### Phase 1: Discover the Impossible Location

Show the raw natural location before construction.

Purpose:
- Create curiosity
- Make the location feel impossible
- Establish the before state

Prompt elements:
- Remote forest
- Cliffside
- Giant rocks
- Moss
- Natural shadows
- Hidden entrance
- Untouched wilderness

---

### Phase 2: Clear and Prepare the Site

The builders begin making the place usable.

Actions:
- Remove rocks
- Cut away small roots
- Sweep dirt
- Flatten the ground
- Clear leaves and debris
- Open up the entrance

Purpose:
- Show the first transformation
- Make the viewer understand that the place can become usable

---

### Phase 3: Dig, Shape, or Expand the Space

The natural structure is adjusted.

Actions:
- Dig into soil
- Shape the cave wall
- Widen the entrance
- Level the interior
- Remove loose stones
- Create enough space for a person to stand or sit

Purpose:
- Turn a raw natural space into a potential room

---

### Phase 4: Build the Entrance

This is the first moment where the space starts to look like a home.

Actions:
- Build a wooden doorway
- Add a small deck
- Install steps
- Add a railing
- Create a visible front entrance

Purpose:
- Change viewer perception from "cave" or "hole" to "house"

---

### Phase 5: Create the Structural Frame

Define the interior architecture.

Actions:
- Install wooden beams
- Build an arched frame
- Add wall framing
- Add ceiling framing
- Create support ribs

Purpose:
- Make the viewer understand the room layout

---

### Phase 6: Install Walls and Ceiling

Turn the frame into an enclosed living space.

Actions:
- Add wooden panels
- Install insulation
- Cover walls
- Build a curved ceiling
- Seal gaps

Purpose:
- Make the space feel protected and livable

---

### Phase 7: Install Flooring

Add the strongest signal of comfort.

Actions:
- Lay wooden flooring
- Install stone tiles
- Smooth the floor
- Add a rug or mat

Purpose:
- Shift the space from construction site to home

---

### Phase 8: Add Lighting and Utilities

Make the space feel modern and usable.

Actions:
- Install warm lights
- Add small ceiling lamps
- Add hidden LED strips
- Install a small power source or cable
- Add a stove or fireplace if needed

Purpose:
- Create warmth and realism

---

### Phase 9: Furnish the Interior

Add human life.

Actions:
- Add bed
- Add small table
- Add chair
- Add shelves
- Add blanket
- Add lanterns
- Add plants or natural decoration

Purpose:
- Make the viewer believe someone could live there

---

### Phase 10: Final Cinematic Reveal

End with a before/after feeling.

Shots:
- Exterior entrance hidden in nature
- Warm light glowing from inside
- Slow camera push-in
- Interior reveal
- Cozy furniture
- Final wide shot of the shelter in nature

Purpose:
- Give emotional payoff
- Make the viewer want to rewatch or share

---

## 4. Universal Prompt Template

```text
Create a vertical 9:16 ultra-realistic AI construction timelapse video.

Concept:
An impossible hidden location in nature is transformed into a cozy livable shelter.

Location:
[Describe the impossible natural location]

Workflow:
1. Discover the untouched hidden natural location.
2. Clear rocks, leaves, dirt, and debris from the site.
3. Dig, shape, and level the interior space.
4. Build a wooden entrance, small deck, steps, and railing.
5. Install wooden structural frames and curved support beams.
6. Add walls, ceiling panels, and natural wood surfaces.
7. Install clean wooden flooring.
8. Add warm lighting, small lamps, and realistic utility details.
9. Furnish the space with a bed, small table, chair, shelves, blanket, and cozy decor.
10. End with a cinematic final reveal showing the hidden shelter glowing warmly in nature.

Visual Style:
Ultra realistic, cinematic, photorealistic, natural lighting, detailed textures, realistic construction process, DIY building timelapse, satisfying transformation, human scale, warm interior, untouched wilderness outside.

Camera Style:
Handheld documentary shots, slow push-ins, close-up construction details, wide establishing shots, smooth cinematic reveal, vertical video composition.

Mood:
Surprising, satisfying, peaceful, cozy, mysterious, hidden-in-nature.

Avoid:
Cartoon style, fantasy magic, unrealistic instant transformation, floating objects, distorted tools, broken anatomy, impossible physics, text overlays, logos, watermarks.
```

## 6. Example Concept 1: Hidden Cliff Crack Shelter

```text
Create a vertical 9:16 ultra-realistic construction timelapse video.

A narrow crack inside a giant mountain cliff is discovered deep in a remote forest. At first, it looks too small and rough to live in. Builders clear rocks, sweep dirt, remove debris, and widen the entrance. Style:
Ultra realistic, cinematic, photorealistic, natural forest lighting, detailed wood textures, satisfying DIY construction timelapse, cozy hidden shelter, realistic human-scale construction.

Camera:
Wide shot of the cliff, close-up of hands clearing stones, timelapse of wooden frame construction, slow push-in through the entrance, final warm interior reveal.

Negative prompt:
No cartoon style, no fantasy magic, no unrealistic instant transformation, no distorted tools, no broken anatomy, no text, no watermark.
```

## 9. Best Video Formula

For repeatable content, use this formula:

```text
Impossible nature location
→ Cleaning
→ Digging / shaping
→ Entrance construction
→ Frame construction
→ Walls and ceiling
→ Flooring
→ Lighting
→ Furniture
→ Final reveal
```

This structure can be reused across many locations while keeping the video fresh by changing only the location and final design.
