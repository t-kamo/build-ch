# Episode001 Hidden Waterfall House — Storyboard v2

## Purpose

Storyboard v2 reduces Veo failure risk by breaking the Hidden Waterfall House build into smaller physical steps.

Current issue:

- duplicate people
- teleporting character
- skipped construction process
- unstable motion continuity

Cause:

One clip was asked to perform too much visual transformation.

Solution:

Use smaller stages, fixed location rules, one identical worker, and physically plausible manual progress.

## Global Continuity Rules

- Keep the same massive waterfall.
- Keep the same dark volcanic rock wall.
- Keep the same forest, mist, and pool.
- Keep the same entrance position.
- Keep the same camera angle unless a stage explicitly allows a slow dolly.
- Only one identical adult male worker may appear.
- Never duplicate the character.
- No teleportation.
- No sudden time jump.
- No sudden body position change.
- One stage equals one small visible construction change.

## Character Lock

One adult male worker only:

- grey hoodie
- black baseball cap
- black work pants
- dark work boots
- black work gloves

He must remain the same person throughout every clip.
